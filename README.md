# Setup Mail Server

# Rocky / Redhat
## Download git-annex:
```
https://git-annex.branchable.com/install/Linux_standalone/
```
## Unzip and link 
```
tar -xvf {}
sudo cp -Ra {} /opt/git-annex
ln -s /opt/git-annex/git-annex /usr/bin/git-annex
```
# Ubuntu
sudo apt update; sudo apt install git-annex
